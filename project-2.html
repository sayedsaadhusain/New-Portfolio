Project Overview

Expense Tracker: Personal Finance Management Tool

Project Overview: Developed a web application, Expense Tracker, to help users manage their personal finances by tracking income and expenses. The application provides a clear overview of financial transactions and helps users maintain a balanced budget.

Skills Used: Frontend: HTML, CSS, JavaScript Backend: Node.js, Express.js Database: MongoDB Other Technologies: RESTful APIs, AJAX Key Features: User Authentication: Implemented secure user authentication using JWT (JSON Web Tokens). Provided features for user registration, login, and profile management. Transaction Management: Users can add, edit, and delete income and expense transactions. Each transaction includes details like date, category, amount, and description. Dashboard and Reports: Developed a dashboard that displays a summary of total income, expenses, and balance. Included visual charts and graphs to represent spending patterns and trends. Responsive Design: Ensured the application is fully responsive and accessible on all devices. Used CSS and Bootstrap to create a flexible and user-friendly layout. Data Persistence: Utilized MongoDB to store user data and transaction records. Implemented CRUD operations (Create, Read, Update, Delete) for managing data.
